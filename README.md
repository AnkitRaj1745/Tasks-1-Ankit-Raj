Repository for Number Game
📌 Project Overview

The Number Game is a simple Java console-based application where the computer generates a random number, and the player tries to guess it within a limited number of attempts. The game provides hints such as whether the guessed number is too high or too low, making it interactive and fun.

🚀 Features
Random number generation
User-friendly console interface
Hint system (Too High / Too Low)
Limited attempts for each round
Score tracking
Option to play multiple rounds
🛠️ Technologies Used
Java
Object-Oriented Programming (OOP)
Java Scanner Class
Random Class
